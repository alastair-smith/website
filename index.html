Web components

About

Web components allow developers to split their websites into encapsulated reusable html components using vanilla javascript.

Components on this page

This page uses 3 components:

  • <hex-link /> - Individual Hex tiles in below example
  • <hex-grid /> - Honeycomb of hex tiles in below example
  • <social-footer /> - Footer at the bottom of the page

Hex Link Component

The <hex-link /> lets me provide a name, a url, and the name of an svg to create a link to that url in the shape of a hexagon with that svg as an image.

In the <head /> of this document I need to have <link rel="import" href="./HexLink/HexLink.html">.

I can then use the following markup to get the hex-link <hex-link name="Example" url="http://www.example.com/" svg="example"></hex-link>.
